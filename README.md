## Banking Stocks Analysis using Yahoo Finance (yfinance)

This project performs a complete quantitative analysis of major Indian and global banking sector stocks using the **yfinance** Python library.  
It includes data extraction, cleaning, visualization, statistical analysis, and return-based performance metrics.

---

### **1. Automatic Data Download**
- Fetches historical OHLCV data for selected banking stocks.
- Configurable tickers, date range, and intervals.

### **2. Data Cleaning & Preprocessing**
- Handles missing values.
- Extracts and stores **closing prices**.
- Computes **daily percentage returns**.
- Resamples prices to **monthly frequency**

### **3. Exploratory Data Analysis (EDA)**
- Line charts of closing prices.
- Volatility comparison across banks.
- Correlation matrix and heatmap of stock returns.

### **4. Statistical & Financial Metrics**
- Daily and monthly returns
- Annualized volatility
- Cumulative returns
- Maximum drawdown
- Sharpe ratio (risk-adjusted performance)

### **5. Visualisations**
- Price evolution plots  
- Daily returns distribution  
- Monthly percentage change bar plot  
- Correlation heatmap  
- Rolling volatility charts
